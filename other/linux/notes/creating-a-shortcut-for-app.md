# Creating a shortcut for app

1. Создать \*.desktop файл в /usr/share/applications
2.  Пример файла: 

```text
[Desktop Entry]
Encoding=UTF-8
Version=1.0
Name=Eclipse
Comment=Eclipse EE IDE
Exec=/home/user/eclipse/eclipse
Icon=/home/user/elcipse/icon.xpm
Terminal=false
Type=Application
Categories=IDE;
```

     3. Сохранить файл
